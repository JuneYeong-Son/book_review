## 완료됨 (참고)
- [x] **신고 버튼**: 서평/토론 신고(`POST /reports`, Report 모델). 관리자 대시보드에서 집계·삭제.
- [x] **마이페이지 내 책 편집**: 책별로 관심 지정 토글 + 서평 날짜별 수정/삭제(`PATCH/DELETE /progress/:id`).
- [x] **관리자 페이지(/admin)**: 오늘의 접속자 수·전체 회원 수·신고된 게시물 수, 신고 많은 게시물 삭제.
      관리자 지정은 env `ADMIN_USERNAMES`(기본 reader). 접속자 집계는 `User.lastSeenAt`.
- [x] **연령대별 추천**: 알라딘이 연령대 미지원 → **우리 플랫폼 자체 데이터**로 연령대별 인기 집계 구현.
      (회원가입/설정에서 출생연도 입력, `method=popular&ageGroup=20` 등)
- [x] **무한 캐러셀**: › 버튼이 끝에 닿으면 서평/토론/베스트셀러를 계속 더 불러옴(페이지네이션).
- [x] **배포**: Render(백엔드+프론트+Postgres). 로컬 SQLite / 배포 Postgres 자동 전환.
- [x] **협업 필터링(CF)**: '읽은 책과 비슷한 책'을 아이템 기반 CF로 대체(+콘텐츠 보충).
- [x] **책 단위 별점**: 서평별 → 책 단위(Rating). 책 상세에서 평균/내 별점.
- [x] **추천 제외(X)**: '추천 안 받을 책' 리스트(RecoExclusion). 특정 책 제외(exclusions).
- [x] **유저 프로필**: `/users/:id`에서 그 유저의 서평·서재·토론.
- [x] **서평 댓글 / 편집·삭제 / URL 개선**(`/books/:bookId/reviews/:번호`).
- [x] **사용자 신고 + 관리자 회원 삭제**.
- [x] **옮긴이만 다른 책 동일 취급**(제목+작가 기준 중복 제거).
- [x] **UI**: 책 이미지 잘림 수정, 책/서평 상세 큰 이미지 레이아웃, 댓글 구분, 내 글귀 탭.

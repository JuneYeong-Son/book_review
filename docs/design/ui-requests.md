# 디자인 / UI 요청 기록

> 사용자가 대화 중 요청한 **디자인·UI 관련** 사항 모음. (전체 개요는 ../requirements.md)

## 헤더
- 상단 네비게이션 바(홈/독서기록/토론 링크)는 **제거**.
- 좌측 "📖 책갈피" 문구, 우측 🔔 알림 버튼 + 프로필(아바타) 동그라미 버튼만.
- 프로필(사진=이모지 아바타) 클릭 → 드롭다운: **내 서평 / 내 책 / 내 토론 / 내 정보 수정 / 로그아웃**.
- 아바타 이미지는 회원가입 시 선택.

## 메인(홈)
- 히어로 문구: **"책을 기록하고, 당신의 세계를 다른 사람과 나눠보세요"**.
- 서평·토론을 **서로 다른 칸**으로 분리해서 표시.
- 서평/토론은 **정사각형 카드 + 가로 캐러셀**, **‹ ›(>) 버튼**으로 넘겨보기.
- 서평 정렬: 좋아요 많거나 내 관심 책 우선. 토론 정렬: 댓글(인기) 많거나 관심 책 우선.
- 내가 쓴 **최신 인상깊은 글귀 2개** 표시.
- 섹션 제목(서평/토론 등) 글씨를 **약 15pt 크게**.
- "책 둘러보기" → **"추천하는 책"** 으로 변경, **🤖 아이콘 제거**.
- 추천도 **캐러셀(> 버튼)** 로 표시.

## 추천 UI
- 추천 방법 **선택(토글)**: ① 읽은 책과 비슷한 책 ② 요즘 많이 사는 책.
- "요즘 많이 사는 책"에 **필터**(장르 등). ※ 연령대(20~30대) 필터는 알라딘 공개 API 미지원.
- 추천 **박스(표지·제목) 클릭 → 그 책의 서평 + 토론** 보기.

## 상세 이동
- **서평 박스 클릭 → 서평 상세 페이지**로 이동.
- 서평 상세에서 **댓글** 표시.
- **홈 서평·토론 카드에서 작성자·책을 개별 클릭**하면 각각 프로필/책 상세로 이동(카드 전체 단일 링크 → 요소별 링크). 서평 댓글 작성자도 프로필 링크.

## 카드 액션 버튼
- 추천 카드의 **‘＋ 내 서재에 추가’** 버튼은 카드마다 **하단 고정·폭 통일**(`.card-action full`), 외부추천 담기 버튼은 가독성을 위해 채운 버튼으로.

## 회원가입 / 계정
- **2단계 회원가입**: ①정보 입력(이메일·닉네임 **실시간 중복확인**·휴대폰·개인정보 동의 체크) → ②이메일로 받은 6자리 코드 입력 → 가입 완료. (메일 미설정 테스트 모드에선 코드 화면 표시)
- **닉네임 = 활동 표시명**: 서평·토론·댓글·프로필·홈 카드 등 활동 표시명은 닉네임(없으면 이름). 설정에서 닉네임 편집.

## 작성/검증 UX
- **서평은 내 서재에 담은 책만** 작성 가능 — 서평 쓰기 모달은 서재 책만 선택, 서재가 비면 안내문.
- **과하게 짧은 글 방지**: 서평 10자↑(빈 값 허용)·토론 내용 10자↑ (`minLength` + 서버 검증).
- 회원가입 비밀번호 안내(8자↑·영문+숫자).

## 토론 상세
- 상단 **책 소개**(표지·제목·저자)와 **토론 본문**을 시각적으로 분리 — 본문은 ‘토론 내용’ 라벨이 붙은 좌측 강조 카드.

## 푸터 / 공통
- **스티키 푸터**: 내용이 짧아도 화면 맨 아래에 고정(푸터가 위로 뜨던 문제 해결).
- 푸터에 **‘💬 의견·버그 신고’** 버튼(모달, 비로그인도 제출) + **이용 약관** 링크.
- **이용 약관** 정적 페이지(`/terms`).
- **관리자 페이지**: 회원 관리(권한·정지·삭제)·피드백 목록(처리·삭제) 섹션.

## 마이페이지
- 탭: 내 서평 / 내 책 / 내 토론 / **달력**.
- 책별로 눌러 들어가면 그 책의 서평을 **날짜별**로.
- **달력 탭**: 그날 무슨 책을 얼마나 읽었는지 간략히 표시.

## 추가·수정된 UI (이후)
- **추천 카드 X 버튼**: 누르면 그 책을 '추천 안 받을 책'에 담아 다시 안 나오고 다른 책으로 교체.
- **추천 방식 정리**: '읽은 책과 비슷한 책'(=CF) / '요즘 많이 사는 책'. 연령대 옵션은 일단 제거.
  필터는 '필터 ▾' 드롭다운 하나로. 추천도 캐러셀로.
- **추천 카드 흐름**: 내 서재에 없으면 '＋ 내 서재에 추가'가 먼저, 담아야 '서평 쓰기' 노출(하트 오버레이 제거).
- **책 이미지 잘림 금지**: 표지는 `object-fit: contain`(캐러셀·썸네일 포함).
- **책 상세 페이지**: 큰 표지 + 오른쪽에 제목·작가·출판사·설명·별점(평균/내 별점). 서평은 좋아요순 정렬.
- **서평 상세 페이지**: 큰 책 이미지 + 옆에 책/작가/작성자. 본인 서평이면 수정/삭제 버튼(인라인 편집).
  서평 본문과 댓글 사이 구분선/여백.
- **마이페이지 탭**: 내 서평(내가 기록한 책) / 내 서재(관심 책 관리, 제거 버튼) / 내 글귀(모아보기) / 내 토론.
  달력은 상단에 항상 표시(달력 탭 없음).
- **유저 이름 클릭 → 프로필**: 서평/토론/댓글의 작성자 이름 클릭 시 유저 프로필로.
- **신고 버튼**: 서평·토론 상세에서 신고, 사용자 신고(작성자 옆).
- **관리자 페이지 `/admin`**: 통계 카드 + 신고 많은 게시물/회원 삭제 (관리자 드롭다운 메뉴).
- **'내 서재에 추가' 모달**: 알라딘 검색으로 새 책 추가만(기존 책 목록 제거, 관리는 내 서재 탭).
- **관심 책 표시**: 관심 책은 갈색 강조. 좋아요(♥)와 혼동 방지를 위해 목록에선 '✕ 제거' 버튼.
